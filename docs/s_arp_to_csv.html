s_arp_to_csv module¶

s_arp_to_csv.script_main(script)¶
SINGLE device script
Author: Jamie Caesar
Email: jcaesar@presidio.com

This script will capture the ARP table of the attached device and output the results as a CSV file. While this script can be used to capture the ARP table, the primary purpose is to create the ARP associations that the “connected_ip.py” script can use to map which MAC and IP addresses are connected to each device.

Parameters:script (script_types.Script) – A subclass of the sessions.Session object that represents this particular script session (either SecureCRTSession or DirectSession)
